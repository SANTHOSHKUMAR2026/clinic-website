Step 1 — Stop Running Server

In the same terminal press:

Ctrl + C

Then press Y if asked.

This stops the old Next.js server.

Step 2 — Kill Stuck Node Process (Important)

Sometimes Windows keeps the process running.

Run this command in PowerShell:

taskkill /F /IM node.exe

This closes all running Node servers.
